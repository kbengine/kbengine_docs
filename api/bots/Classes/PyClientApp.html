详细描述

类Entity的实例代表着在client上的游戏对象。



一个Entity可以通过ENTITYCALL访问在base和cell应用程序上的等价的实体。这需要 一组远程调用的函数（在实体的.def文件里指定）。

成员函数文档

def getSpaceData( key ):

功能说明：

获取指定key的space数据。
space数据由用户在服务端通过setSpaceData设置。

参数：
key string，一个字符串关键字。

返回：
string，指定key的字符串数据。

def player( ):

功能说明：

获得当前客户端所控制的实体。
